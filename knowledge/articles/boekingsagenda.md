# Waar vind ik mijn persoonlijke boekingsagenda?

In dit artikel lees je waar je je geboekte lesdagen, workshops en themadagen kunt terugvinden.

## Link in je bevestigingsmail

Na het boeken van een lesmodule, workshop of themadag ontvang je een bevestigingsmail. In deze mail staat een link naar je persoonlijke boekingsagenda. Bewaar deze mail en de link goed -- sla de link op als bladwijzer.

## Wat zie je in je boekingsagenda?

In je persoonlijke boekingsagenda vind je een actueel overzicht van al je boekingen. Je ziet daar alle lesdagen, workshops en themadagen die je hebt geboekt, eventuele wijzigingen in data en nieuwe boekingen die je later toevoegt (zoals vervolgmodules, een praktijkexamen of workshops). Deze worden automatisch aan je agenda toegevoegd.

## Alleen toegankelijk via de link

Je persoonlijke boekingsagenda is alleen toegankelijk via de link in je bevestigingsmail. Er is geen apart dashboard of account waarin je deze agenda kunt terugvinden. Bewaar je bevestigingsmail daarom goed.
