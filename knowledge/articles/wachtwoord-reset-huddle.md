# Hoe reset ik mijn wachtwoord op Huddle?

In dit artikel lees je hoe je je wachtwoord op Huddle opnieuw kunt instellen.

## Wachtwoord vergeten

1. Ga naar de inlogpagina van [Huddle](https://sanayou.thehuddle.nl/)
2. Klik op "Wachtwoord vergeten"
3. Vul het e-mailadres in waarmee je bent aangemeld
4. Je ontvangt een herstelmail waarmee je een nieuw wachtwoord kunt instellen

## Geen herstelmail ontvangen?

Controleer je spammap als de mail niet meteen aankomt. Komt de mail ook daar niet binnen, dan is je account mogelijk gekoppeld aan een ander e-mailadres. Neem in dat geval contact op via de chat rechtsonder op de pagina.

## Alternatief: inloggen via een magische link

Huddle biedt ook de optie om in te loggen zonder wachtwoord. Op de inlogpagina kun je kiezen voor inloggen via een magische link. Je vult je e-mailadres in en ontvangt een eenmalige inloglink per e-mail. De optie voor wachtwoord-inlog staat direct eronder.
